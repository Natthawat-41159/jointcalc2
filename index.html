ℹ About JointCalc Pro v0.5.1

Phase 4 · β — Steel on RC Column

Hybrid steel-to-concrete connection design tool with full member orientation flexibility.

What's new in v0.5.1 — Mobile Responsive + Canopy Report (TH)

  • Auto mobile-mode — เปิด mobile layout อัตโนมัติบน phone/tablet (≤1024px) และ touch devices
  • Resize listener — สลับ desktop↔mobile เมื่อ resize หน้าต่าง / หมุนจอ (orientationchange)
  • Touch targets — input, select, section-head สูงขึ้น (min 40-44px) ตาม WCAG 2.5.5
  • Header tabs scroll — connection tabs scroll แนวนอนบน phone ไม่ overflow
  • Phone (≤480px) — label แคบลง, template grid 2-col, UI กระชับขึ้น
  • iOS safe-area — tab bar และ sticky calc button หลีกเลี่ยง home indicator
  • 3D viewport resize fix — double-trigger onResize หลัง tab switch แก้ iOS Safari repaint
  • Canopy calc sheet ภาษาไทย — รายการคำนวณ Wall/Col Canopy แปลเป็นภาษาไทยทั้งหมด ดึง data path ถูกต้อง แสดง DCR bar + หมวดหมู่ (โครงสร้าง/ยึดเหนี่ยว/รอยเชื่อม) พร้อม geometry แยก wall/col

What's new in v0.5.0 — Sprint A: Anchor Formula Audit

  • [FIX] ANc edge-aware projection — §17.6.2.1: clip ด้วย ca จริงแต่ละด้าน → conservative เมื่อ anchor ใกล้ขอบ
  • [ADD] ψec_N eccentricity factor — §17.6.2.4: ψec = 1/(1+2eN/3hef) ≤ 1.0
  • [FIX] AVc height limit (ha) — §17.7.2.1: min(1.5ca, ha=colH) ไม่ใช่ colH/2
  • [ADD] ψed_V edge factor shear — §17.7.2.5: ψed_V = 0.7+0.3ca2/(1.5ca)
  • [UI] eqn string — แสดง ψec, ψed_V, ha ใน result row

What's new in v0.4.9

  • Stiffener design — Flat bar / gusset input + plate bending + weld check (AISC 360 §F11, §J2.4) · 3D visualization
  • J-Bolt option — Pullout = hook bearing + bond strength + heavy washer factor · 3D hook visualization
  • Conservative fc' toggle — ป้อน fc' จริง → แอปลด 20% อัตโนมัติก่อนคำนวณ
  • Anchor pattern selector — 2×2, 3×2, 2×3, 2×4, 4×2 พร้อม breakout area ที่ถูกต้อง

What's new in v0.4.7

  • 📋 Paste from Prota / Excel — paste member forces table → auto-fill
  • Single + Envelope modes

What's new in v0.4.6

  • Manual offset — กรอกค่า Offset X/Y/Z ต่อ member
  • Snap to Plate ⏬ + Reset ↺
  • Cut Style — Flat / Perpendicular / Mitre

What's new in v0.4.5

  • Member orientation fix — Top flange ชี้ขึ้นเสมอ (เดิม section หมุนรอบ axis ไม่คุ้มที่)
  • ใช้ orthonormal basis (X→dir, Y→up) แทน quaternion shortest-path
  • Local origin ของ section ย้ายมาที่ bottom flange surface → bottom flange นั่งบน plate

What's new in v0.4.4

  • Trim fix — แก้บั๊กคานหลุดออกจาก base plate
  • ปลายคาน (near end) วางที่ attach point พอดี — ไม่มี pushBack

What's new in v0.4.3

  • Auto-trim ends ✂ — ปลายคานตัดเรียบกับหน้า base plate (เหมือนงาน fab จริง)
  • ใช้ Three.js clipping planes ตัดปลายคานตามหน้า plate/column
  • ปุ่ม ✂ ใน viewport toolbar เปิด/ปิดได้
  • ลบการอ้างอิงชื่อแบรนด์ third-party ออกทั้งหมด

What's new in v0.4.2

  • Bug fix #2 — 3D member placement ใช้ quaternion math ให้ติด base plate ถูกต้องทุก template
  • Tooltips (?) — กดที่ ? เพื่อดูคำอธิบายของแต่ละช่อง พร้อมตัวอย่างค่า
  • Side-attached members — รองรับคานยื่นจากด้านข้างเสาแล้ว (template F/G/H/I)

What's new in v0.4.1

  • Slope mode — กรอกมุมแบบ slope (จากแนวราบ) ตามแบบหลังคา
  • Quick presets — 0°/15°/20°/30°/45° กดปุ่มได้
  • Smart suggestions — เมื่อ DCR FAIL แอปแนะนำวิธีแก้

Features (from v0.4.0)

  • 9 configuration templates + custom angles — handle any real-world orientation
  • Multiple members meeting at one column with independent angles & loads
  • ACI 318-19 Chapter 17 anchor design engine — 5 limit states
  • AISC DG-1 base plate (small + large eccentricity)
  • Concrete bearing per ACI 318 §22.8
  • Force resolution — auto-decomposes member axial/shear into vertical & horizontal at base plate

Engineering basis

  • Anchor steel: ACI 318-19 §17.6.1, §17.7.1
  • Concrete breakout: ACI 318-19 §17.6.2 (tension), §17.7.2 (shear)
  • Pullout: ACI 318-19 §17.6.3
  • Side-face blowout: ACI 318-19 §17.6.4
  • Pry-out: ACI 318-19 §17.7.3
  • Combined T+V: ACI 318-19 §17.8
  • Base plate bending: AISC DG-1 §3
  • Concrete bearing: ACI 318-19 §22.8

⚠ Limitations

  • Group effects assume rectangular pattern
  • Concrete edge cases (corner anchors) use conservative ψ factors
  • Anchor reinforcement not yet computed (toggle changes φ only)
  • No seismic provisions yet (Phase 2 will add AISC 341)
